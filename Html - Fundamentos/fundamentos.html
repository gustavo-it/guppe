Lista ordenadas. Indica uma lista ordenada por números ou letras. para isso utilizamos a tag "ol", para criar a lista e "li" para indicar os itens. Podemos passar um atributo "type" para a tag ol. para indicarmos o tipo de ordenação que queremos

  1. type="1" - Lista por números;
  2. type="A" - Lista por letras maiúsculas;
  3. type="a" - Lista por letras minúsculas;
  4. type="I" - Lista por números romanos maiúsculos;
  5. type="i" - Lista por números romanos minúsculos.

Links e âncoras

Podemos linkar uma página a outra, ou até mesmo fazer referências através da tag "a". Para isso passamos o nome do arquivo ou link via "href=". Podemos utilizar o valor "target="_blank" para abrir em uma nova janela. link do google

Se quisermos fazer download de algum arquivo, podemos passar o endereço do arquivo em href e passar para a tag "a" o atributo download.

Outro uso da datg "a" é a criação de âncoras na página.sso faz com que quando o usuário estiver em uma página muito grande, ao clicar em um link a página irá rolar até o conteúdo marcado com a âncora. Para isso, basta passar um atributo “id” com um nome para o elemento que se quer localizar, e passar esse nome no href com “#”. Voltar ao topo da página.
